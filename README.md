## Installation

You can install the development version of biolingR like so:

``` r
# install.packages("remotes")
remotes::install_github("stefanocoretta/biolingR")
```

## Usage

To load the data:

``` r
library(biolingR)
data("bii_nlangs")
?bii_nlangs
```
